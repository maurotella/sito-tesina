Cos'è django?

Django è un web framework scritto in Python.

Cos'è un web framework?

Un web framework è un insieme di componenti (struttura) progettati per semplificare lo sviluppo di applicazioni e siti web.

Perchè si usa un web framework?

I web framework hanno trasformato il mondo dello sviluppo di applicazioni o siti web, in particolar modo per quanto riguarda i tempi di sviluppo in quanto il framework fornisce già allo sviluppatore tutti i componenti necessari allo sviluppo. Non ci sarà quindi bisogno di perdere tempo con la scrittura di codice e con i possibili problemi che essa comporta. Tra i componenti che un web framework offre ci sono tutti quelli che permottono al sito di funzionare su un web server e interagire con i meccanismi che sono alla base del funzionamento di Internet stesso.
