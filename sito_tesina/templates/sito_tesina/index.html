Cos'è django?

Django è un web framework scritto in Python.

Cos'è un web framework?

Un web framework è un framework (struttura) software progettato per semplificare lo sviluppo di applicazioni e siti web.

Perchè c'è bisogno di un web framework?

  • Sito/Applicazione web dinamico/a;
  • Struttura preesistente che riduce le conoscenze e il tempo necessari allo sviluppo.
